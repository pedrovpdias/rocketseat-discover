# Event Module

É um mecanismo para
* [ ] Disparar eventos
* [ ] Ouvir eventos
* [ ] Fazer alguma ação quando ouvir o evento
* [ ] É a base para muitos outos módulos como:
**  [] http
**  [] stream
**  [] file
**  [] system
**  [] outros...

## O que vamos aprender?

* [ ] Utilizar os eventos
* [ ] Disparar eventos
* [ ] Ouvir eventos
* [ ] Executar ações para determinados eventos
* [ ] Entender como ele é a base para os outros módulos
